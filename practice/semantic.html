Instuctions on how to open and close vim

  1. Open your terminal.
  2. Type ls.
  3. Then Type cd Desktop
  4. After that use cd and ls to pull up the file you want to vim
  5. Type vim filename and press Enter
  6. Now you can edit your file using i to type things.
  7. to exit vim:
    • Press Esc to make sure you are not typing in text
    • Type:wq and then hit Enter to save and quit

Here is an example

You type: vim index.html

The computer responds with: index.html [New File]
